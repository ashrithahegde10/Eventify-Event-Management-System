# Eventify-Event-Management-System
Eventify is a MERN-based event management platform featuring user authentication, event creation, media uploads, and event discovery.


# Eventify – Event Management Platform

Eventify is a full-stack web application that enables users to discover, create, and manage events efficiently. Organizers can publish and manage events, while users can explore upcoming events and save those they are interested in attending.

## Features
- User authentication and authorization using JWT
- Create, update, and manage events
- Upload event media (images/videos)
- Save and track favorite events
- Automatic archiving of past events
- Location-based event filtering
- Responsive and user-friendly interface

## Tech Stack
**Frontend**
- React
- React Router
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB with Mongoose

**Other Tools**
- JWT for authentication
- Multer for file uploads
- bcrypt for password hashing

## Project Purpose
This project was developed as part of a **Web Technologies academic project** to demonstrate full-stack development skills, RESTful API design, authentication, file handling, and database integration.
